This project is a roguelike that can be run in the terminal. It is 1 wave of 1v1 monster fights, with 1 types of monster battle available. More features were planned but i ran out of time.
You are given base stats and a mech suit that adds to your stats. Your mech suit is what is going to be upgraded over time, and you can choose what kind of base stats you have at the begininng to suit your playstyle.

# How to compile:
javac theGame.java

# How to run:
java theGame
